# Real-World Data Project: Retail Sales Analysis & Prediction

## 📌 Objective
Analyze retail sales data and build a simple prediction model.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Features
- Data cleaning
- Sales trend analysis
- Visualization (heatmaps, line charts, bar charts)
- Correlation analysis
- Sales prediction using Linear Regression

## ▶️ How to Run
pip install -r requirements.txt
python retail_analysis.py

## 📁 Files
- retail_analysis.py
- retail_data.csv
- requirements.txt

## 📈 Output
- sales_trend.png
- correlation_heatmap.png
- prediction_plot.png

## 💡 Outcome
Understand real-world data patterns and build prediction models.
